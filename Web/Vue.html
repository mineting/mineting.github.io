                                                                                          # 介绍

                                                                                          Vue 是一款用于构建用户界面的 JavaScript 框架。

                                                                                          它基于标准 HTML、CSS 和 JavaScript 构建，并提供了一套声明式的、组件化的编程模型，帮助你高效地开发用户界面。

                                                                                          # 特点

                                                                                          • 遵循 MVVM 模型

                                                                                          • 组件化编码，提高代码复用率

                                                                                          • 声明式编码，直接操作 DOM

                                                                                          • 使用虚拟 DOM + 优秀的 DIFF 算法，尽量复用 DOM 节点

                                                                                          # 与其它 JS 框架的关联

                                                                                          • 借鉴 Angular 的模板和数据绑定技术
                                                                                          • 借鉴 React 的组件化和虚拟 DOM 技术

                                                                                          # GettingStarted

                                                                                          注意 js 表达式与 js 代码的区别：

                                                                                          • js 表达式：可以用一个具体的值接收，如：a、a + b、fun(a)、x >= y ? 0 : 1 以及 .toUpperCase()等
                                                                                          • js 代码：if(){}、for(){}
                                                                                          <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>GettingStarted</title>
    <script src="./js/vue.js"></script>
</head>
<body>
    <!-- 
        注意事项：
        1. Vue实例和容器是一一对应的，多个容器对应一个实例，Vue只会管理第一个容器，
            多个实例管理一个容器，则只有第一个实例会生效，多个实例也可以一一管理多个容器
        2. 插值表达式中应该写 js 表达式，且属性为 data 中的属性
            js 表达式：可以用一个具体的值接收，如：a、a + b、fun(a)、x >= y ? 0 : 1 以及 .toUpperCase()等
            js 代码：if(){}、for(){}
     -->
    <div id="root">
        <h1>Hello {{message}}!</h1>
    </div>
    <script>
        // 设置为 false 以阻止 vue 在启动时生成生产提示。
        Vue.config.productionTip = false; 

        // Vue 对象中传入配置对象：$options
        const vm = new Vue({
            //element：指定当前实例为哪一个容器服务，值为CSS选择器
            //el: document.getElementById("root"),
            el: "#root",
            //data 指定为 el 服务容器数据
            data: {
                message:'Vue'
            }
        })
    </script>
</body>
</html>

                                                                                          # 模板语法

                                                                                          • 插值语法
                                                                                            • 用于解析标签体内容
                                                                                            • 写法：，js 表达式，且可以读取到 data 中的属性
                                                                                          • 指令语法
                                                                                            • 用于解析标签属性、绑定事件等
                                                                                            • 写法：v-bind:href="url" 或者简写为 :href="url", 可以写 js 表达式以及读取 data 中的属性

                                                                                          使用示例：

                                                                                          <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>模板语法</title>
    <script src="./js/vue.js"></script>
</head>
<body>
    <!-- 
        模板语法分为：插值语法和指令语法
        1. 插值语法：用于解析标签体内容
            写法：{{data}}，js 表达式，且可以读取到 data 中的属性
        2. 指令语法：用于解析标签属性、绑定事件等
            写法：v-bind:href="url" 或者简写为 :href="url", 可以写 js 表达式以及读取 data 中的属性
     -->
    <div id="root">
        <h1>插值语法</h1>
        <h2>Hello {{message}}!</h2>
        
        <h1>指令语法</h1>
        <!-- v-bind：属性绑定 -->
        <!-- 模板可以直接读取 this 上的属性，而不用 this.XXX -->
        <a v-bind:href="info.url.toUpperCase()" :x="info.math">指令语法测试</a>
        <a v-bind:href="Date.now()" :x="info.message">指令语法测试</a>
    </div>
    <script>
        Vue.config.prodcutionTip = false
        new Vue({
            el:'#root',
            data:{
                message: 'World',
                info: {
                    url: 'MineTing',
                    math: 'doublex',
                    message: 'Example'
                }
            }
        })
    </script>
</body>
</html>

                                                                                          # 数据绑定

                                                                                          两种数据绑定方式：

                                                                                          1. 单向数据绑定：v-bind, 数据只能从 data 流向页面
                                                                                          2. 双向数据绑定：v-model, 数据从 data 流向页面或者从页面流向 data

                                                                                          ​ （1）双向绑定一般应用在表单元素上，如 h2 不可以

                                                                                          ​ （2）v-model:value 可以简写为 v-model, 默认收集 value 值

                                                                                          示例：

                                                                                          <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>数据绑定</title>
    <script src="./js/vue.js"></script>
</head>

<body>
    <!-- 
        两种数据绑定方式：
        1. 单向数据绑定：v-bind, 数据只能从 data 流向页面
        2. 双向数据绑定：v-model, 数据从 data 流向页面或者从页面流向 data
            （1）双向绑定一般应用在表单元素上，如 h2 不可以
            （2）v-model:value 可以简写为 v-model, 默认收集 value 值
     -->
    <div id="root">
        <h1>数据绑定</h1>
        单向数据绑定：<input type="text" :value="name" /><br />
        双向数据绑定：<input type="text" v-model:value="name" /><br />
        双向数据绑定：<input type="text" v-model="name" />
        <!-- 错误写法：h2 标签没有value -->
        <!-- <h2 v-model:x="name">错误写法</h2> -->
    </div>
    <script>
        Vue.config.prodcutionTip = false
        new Vue({
            el: '#root',
            data: {
                name: 'World'
            }
        })
        for(var i = 0;i<100;i++){
            console.log("我爱宝宝");
        }
    </script>
</body>

</html>

                                                                                          # el 与 data 的两种写法

                                                                                          <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>el 与 data 的两种写法</title>
    <script src="./js/vue.js"></script>
</head>
<body>
    <!-- 
        el 与 data 的两种写法：
            el：
                （1）new Vue() 时配置 el：指定选择器或者 document.getElementById
                （2）先创建好 Vue 实例，后挂载 el, vm.$mount('#root');
            data：
                （1）对象式
                （2）函数式（组件使用）
                    注意：不能写成箭头函数，Vue去调用该函数,this 是Vue实例
     -->
    <div id="root">
        <h1>Hello {{message}}!</h1>
    </div>
    <script>
        Vue.config.productionTip = false; 

        const vm = new Vue({
            // el 的第一种写法
            // el: "#root",

            // data的第一种写法：对象式
            // data: {
            //     message:'Vue'
            // }

            // data的第二种写法：函数式
            // data:function(){
            data(){
                //此处的 this 是 Vue 实例对象，不能写成箭头函数，否则 this 是 Window 对象
                console.log(this); 
                return {
                    message: 'Vue'
                }
            }
        });
        console.log(vm);

        setTimeout(() => {
            //el 第二种写法：更加灵活
            vm.$mount('#root');
        }, 1000);
    </script>
</body>
</html>

                                                                                          # MVVM 模型

                                                                                          • M：模型（Model） ：对应 data 中的数据
                                                                                          • V：视图（View）：模板
                                                                                          • VM：视图模型（ViewModel） ： Vue 实例对象

                                                                                          MVVM 模型：

                                                                                          <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>MVVM模型</title>
    <script src="./js/vue.js"></script>
</head>

<body>
    <!-- 
        M：Model，对应 data 中的数据
        V：View，视图，对应模板
        VM：ViewModel，对应 Vue 实例对象

        说明：
            （1）data 上的所有属性都会出现在 vm 上
            （2）vm 上的所有属性以及 Vue 上的所有属性，在 Vue 模板上都可以直接调用，如 _c、$emit等
     -->
    <div id="root">
        <!-- 此处相当于 View -->
        <h1>Name: {{name}}</h1>
        <h1>Address: {{address}}</h1>
        <h1>Address: {{_c}}</h1>
    </div>

    <script>
        Vue.config.productionTip = false;

        // 此处相当于 ViewModel，经常用 vm 代替该实例
        const vm = new Vue({
            el: "#root",
            // 此处相当于 Model
            data: {
                name: 'Sample',
                address: 'Xian'
            }
        });
        console.log(vm);
        
    </script>
</body>

</html>

                                                                                          # Object.defineProperty

                                                                                          <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Object.defineProperty</title>
    <script src="./js/vue.js"></script>

</head>

<body>
    <!-- 
        由 Object.defineProperty 设置对象的值：
            默认不能被枚举、修改、删除，需要进行配置
            // value: 13,
            // enumerable: true,   //控制属性是否可以被枚举 FALSE
            // writable: true,  //控制属性是否可以被修改，默认为 FALSE
            // configurable: true,  //控制属性是否可以被删除，默认为 FALSE
            
            其中 get 和 set 方法可以实现数据的双向绑定
     -->
    <script>

        let number = 8;

        let person = {
            name: "tom",
            sex: 'Man',
            // age: 12,
            // 此处设置的 number 只会赋值一次，number 改变时，person 对象的 age 属性值不会改变
            age: number
        }

        //注意属性的引号
        Object.defineProperty(person, 'age', {
            // value: 13,
            // enumerable: true,   // 控制属性是否可以被枚举 FALSE
            // writable: true,     // 控制属性是否可以被修改，默认为 FALSE
            // configurable: true,  // 控制属性是否可以被删除，默认为 FALSE

            // 读取 person.age 属性值时，会调用该方法，返回值即为age的值
            get: function () {
                console.log("getter....");
                return number;
            },
            // 设置 person.age 属性值时，会调用该方法
            set(value) {
                console.log("setter....value: ", value);
                // 绑定number和value
                number = value
            }
        })

        // 遍历对象的 key
        console.log(Object.keys(person));

        for (let key in person) {
            console.log(person[key]);
        }

        console.log(person);

    </script>
</body>

</html>

                                                                                          <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">

    <!-- 
        Vue 中的数据代理：
        （1）通过 vm 对象来代理 data 对象中属性的操作
        （2）可以更加方便的操作 data 中数据
        （3）基本原理：
            vm.data = undefined; Vue 会将 data 中的值存储在 _data 中，并额外的在 _data 上通过观察者模式添加与页面的响应
            通过 Object.defineProperty() 把 _data 上的所有属性添加到 vm 上, 如 vm.name, vm.address
            数据代理也简化了 _data.name 为 name
            为每一个添加到 vm 上的属性，都指定一个 getter/setter
            在 getter/setter 内部去操作 data 中的属性

    <div id="root">
        <h1>Name: {{name}}</h1>
        <h1>Name: {{_data.name}}</h1>
        <h1>Address: {{address}}</h1>
    </div>

    <script>
        Vue.config.productionTip = false;

        let data = {
            name: 'Sample',
            address: 'Xian'
        };
        // vm._data = $options.data = data
        const vm = new Vue({
            el: "#root",
            data
        });
        console.log(vm);
    </script>
</body>

</html>

                                                                                          # 事件的基本使用

                                                                                          <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>事件的基本使用</title>
    <script src="./js/vue.js"></script>
</head>

<body>
    <!-- 
        事件的基本使用：
        1. 使用 v-on:事件名 或者简写为 @事件名
        2. 事件的回调配置在 methods 上，最终显示在 vm 上
        3. methods 中配置的函数不要使用箭头函数，this 应指向 vm
        4. 函数名后可以加或不加括号(默认传递事件对象)，如果传参需要添加括号（不会默认传递事件对象 $event）
        5. 如果在插值语法中使用函数：不加括号会直接返回函数定义信息，添加括号才会得到返回值
     -->
    <div id="root">
        <h1>测试事件的基本使用</h1>
        <!-- 函数名后可以加或不加括号(默认传递事件对象)，如果传参需要添加（不会默认传递事件对象） -->
        <button v-on:click="showInfo1">事件一（不传参）</button>
        <!-- $event：可以看做 Vue 的关键字(内置对象)，表示事件对象，与参数顺序无关，可以放在前面或者后面 -->
        <button @click="showInfo2(2,$event)">事件二（传参）</button>
        <h1>函数值：{{showInfo3()}}</h1>
    </div>

    <script>
        Vue.config.productionTip = false;

        const vm = new Vue({
            el: "#root",
            // 只有 data 中的属性/函数才会进行数据劫持,生成 getter/setter
            data: {
                // 不建议函数写在这里
                func(){
                    console.log("data 中的函数也会进行数据劫持，而 methods 中的不会");
                }
            },
            methods: {
                // 注意不要使用箭头函数
                // showInfo1:()=>{}
                // showInfo1(event) {
                showInfo1(event) {
                    console.log("showInfo1...",event.target.innerHTML);
                },
                showInfo2(num, event) {
                    console.log(num, event.target.innerHTML);
                },
                showInfo3(event){
                    console.log(event); // undefined
                    return "return";
                }
            }
        });
        console.log(vm);
    </script>
</body>

</html>

                                                                                          # 事件修饰符

                                                                                          <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>事件修饰符</title>
    <script src="./js/vue.js"></script>
    <style>
        .demo1 {
            margin: 20px 0 20px;
            height: 50px;
            background-color: aqua;
        }

        .box1 {
            padding: 5px;
            background-color: skyblue;
        }

        .box2 {
            padding: 5px;
            background-color: orange;
        }

        .list {
            width: 200px;
            height: 200px;
            background-color: greenyellow;
            overflow: auto;
        }

        li {
            height: 100px;
        }
    </style>
</head>

<body>
    <!-- 
        Vue 事件修改符：
        0. 使用 v-on:click="" 为元素绑定事件，可以简化为 @click=""
        1. prevent：阻止默认事件
        2. stop：阻止事件冒泡
        3. once：之间只触发一次
        4. capture：使用事件的捕获模式,在捕获阶段（对于嵌套的元素来说，右外往内执行事件）进行事件的执行，而不是在冒泡阶段（由内往外执行）
        5. self：只有event.target是当前操作元素才会触发事件
        6. passive：事件的默认行为无需等待事件回调执行完毕
     -->

                                                                                          # 计算属性示例

                                                                                          <!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>姓名案例</title>
    <script src="./js/vue.js"></script>
</head>

<body>
    <!-- 
        计算属性：
            1. 要用的属性不存在（如果 data 中有属性则会报错），要通过已有属性【被 Vue 管理的属性】计算得来，也是属性
            2. 原理：底层借助了 Objcet.defineproperty 方法提供的 getter 和 setter
            3. get函数执行时机
                        (1).初次读取时会执行一次
                        (2).当依赖的数据发生改变时会被再次调用
            4. 优势：与 methods 实现相比，内部有缓存机制（复用），效率更高，调试方便
            5. 注意事项：
                    （1）计算属性最终会出现在 vm 上，直接读取使用即可
                    （2）如果计算属性要被修改，那必须写 set 函数去响应修改，且 set 中要引起计算时依赖的数据发生改变
            6. 计算属性的简写形式：
                不考虑修改时，只需要getter
     -->
    <div id="root">
        <h2>计算属性实现</h2>
        姓：<input type="text" v-model:value="firstName" /><br>
        名：<input type="text" v-model="lastName" /><br>
        姓名：<span>{{fullName}}</span>
        <!-- 计算属性会进行缓存，多个结果只会计算一个，methods 每次都会调用一次方法 -->
        姓名：<span>{{fullName}}</span>
        姓名：<span>{{fullName}}</span>
        姓名：<span>{{fullName}}</span>
    </div>
    <script>
        const vm = new Vue({
            el: "#root",
            data: {
                firstName: "",
                lastName: "",
            },
            methods:{
                // fullName(){
                //     return this.firstName + this.lastName
                // }
            },
            computed: {
                // vm._data.firstName 以及 vm._data.lastName 都不为undefined，但是计算属性的 vm._data.fullName = undefined
                fullName: {
                    // 读取 fullName 时，get 的返回值会作为 fullName 的值
                    // get 的调用时机：1. 初次读取；2. 所依赖的属性发生变化
                    // 不要写成箭头函数
                    get() {
                        console.log("fullName get...");
                        // console.log(this); // vm
                        return this.firstName + "-" + this.lastName;
                    },
                    // 当 fullName 被修改时调用,一般不会修改
                    set(value) {
                        console.log("fullName set...");
                        const arr = value.split('-');
                        this.firstName = arr[0]
                        this.lastName = arr[1]
                    }
                },
                
                // 简写形式：更多情况下仅仅获取而不进行修改
                // fullName2:function() {
                fullName2() {
                    return this.firstName + "-" + this.lastName;
                }
            }
        })
    </script>
</body>

</html>

        4. capture：使用事件的捕获模式,在捕获阶段进行事件的执行，而不是在冒泡阶段

                                                                                          <!DOCTYPE html>

<body>
    <!-- 
        计算属性：
            1. 要用的属性不存在，要通过已有属性计算得来，也是属性
            2. 原理：底层借助了 Objcet.defineproperty 方法提供的 getter 和 setter
            3. get函数执行时机
                        (1).初次读取时会执行一次
                        (2).当依赖的数据发生改变时会被再次调用
            4. 优势：与methods实现相比，内部有缓存机制（复用），效率更高，调试方便
            5. 注意事项：
                    （1）计算属性最终会出现在 vm 上，直接读取使用即可
                    （2）如果计算属性要被修改，那必须写 set 函数去响应修改，且 set 中要引起计算时依赖的数据发生改变
            6. 计算属性的简写形式：
                不考虑修改时，只需要getter
     -->
    <div id="root">
        <h2>计算属性实现</h2>
        姓：<input type="text" v-model:value="firstName" /><br>
        名：<input type="text" v-model="lastName" /><br>
        姓名：<span>{{fullName}}</span>
        <!-- 计算属性会进行缓存 -->
        姓名：<span>{{fullName}}</span>
        姓名：<span>{{fullName}}</span>
    </div>
    <script>
        const vm = new Vue({
            el: "#root",
            data: {
                firstName: "",
                lastName: "",
            },
            computed: {
                // vm._data.fullName = undefined
                fullName: {
                    // 读取 fullName 时，get 的返回值会作为 fullName 的值
                    // get 的调用时机：1. 初次读取；2. 所依赖的属性发生变化
                    // 不要写成箭头函数
                    get() {
                        console.log("fullName get...");
                        // console.log(this);
                        return this.firstName + "-" + this.lastName;
                    },
                    // 当 fullName 被修改时调用,一般不会修改
                    set(value) {
                        console.log("fullName set...");
                        const arr = value.split('-');
                        this.firstName = arr[0]
                        this.lastName = arr[1]
                    }
                },
                
                // 简写形式
                // fullName2:function() {
                fullName2() {
                    return this.firstName + "-" + this.lastName;
                }
            }
        })
    </script>
