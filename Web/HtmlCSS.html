    # 网页概述

    ​ HTML：超文本标记语言（HyperText Markup Language）

    网页结构：

    一个网页由三部分组成，结构、表现、行为。

    结构：使用HTML编写，页面的整体的结构，如标题、段落、图片、音视频等。

    表现：使用CSS编写，页面的外在样式，字体、颜色、背景等。

    行为：使用JavaScript编写，页面与用户之间的交互行为。

    一般要求结构、表现、行为三者分离。

    HTML使用标签的的形式来标识网页中的不同组成部分。

    网页的基本结构：

    <!--DOCTYPE声明HTML的版本-->
<!DOCTYPE html>
<!--html标签不区分大小写，尽量全部小写-->
<html>
	<!--head标签,是对网页元数据的说明，一个网页只能有一个标签-->
	<head>
		<meta charset="UTF-8">
		<!--title标签中的文字，是页面优化的最重要因素。在搜索引擎的搜索时最先看到的、最醒目的内容-->
		<title>HTML入门示例</title>
		<!--使用meta标签可以设置网页的关键字-->
		<meta name="keywords" content="HTML，CSS，JavaScript，jQuery,Vue"/>
		<!--指定网页的描述，搜索引擎在检索页面时，会同时检索页面中的关键词和描述，但是这两个值不会影响页面在搜索引擎中的排名-->
		<meta name="description" content="HTML页面的基本示例">
		<!--
			使用meta可以用来做请求的重定向,指定秒数之后会跳转到指定的目标路径。
			<meta http-equiv="refresh" content="秒数;url=目标路径" />
		-->
		<meta http-equiv="refresh" content="3;url=http://www.baidu.com">
	</head>
	<body>
		<!--body标签中是HTML页面的内容部分-->
		<!--h1标签：标题标签，是搜索引擎检索的重要标签，重要性仅次于title-->
		<h1>Hello World!</h1>
	</body>
</html>

    # 链接标签
    <a href="跳转目标" target="目标窗口的弹出方式">文本或图像</a>

    href：指定链接目标的 url 地址。
    target：指定链接页面的打开方式，其取值有 self 和 blank 两种，其中 _self 为默认值，__blank为在新窗口中打开方式。
    ​ 同时可以设置一个内联框架的 name 属性值，链接将会在指定的内联框架中打开。
    锚点定位：通过创建锚点链接，用户能够快速定位到目标内容。
    可以分为两种方式：一种使用id和id选择器，另一种使用name和target跳转。
    示例：
    <!--
	1.	使用相应的id名标注跳转目标的位置。
	2.	使用<a href="#id名">链接文本</a>创建链接文本。
-->
<h3 id="head">HEAD</h3>
<h3 name="head">HEAD</h3>

<a href="#head">To HEAD
<a href="#" target="head ">To HEAD

    # 表格
    作用：显示数据。包括 <table>、<tr>、<td>、<th>、<caption>、<thead>、<tbody>、<tfoot>
    ​ 首先<caption>和<th>分别表示表格的标题和表头，<th> 一般表头单元格位于表格的第一行或第一列，并且文本加粗居中。
    对于比较复杂的表格，又将表格分割成三个部分：题头、正文和脚注。分别用： thead，tbody，tfoot来标注，这样更好的分清表格结构。
    <thead>： 用于定义表格的头部。用来放标题之类的东西。<thead> 内部必须拥有<tr> 标签
    <tbody>：用于定义表格的主体。
    <tfoot>：放表格的脚注之类。
    表格属性:
    属性名含义
    border设置边框
    bordercolor边框颜色
    cellspacing设置单元格与单元格边框之间的距离，默认2像素。
    cellpadding设置单元格内容与单元格之间的距离，默认1像素。
    width设置表格的宽度
    height设置表格的高度
    aligntd：单元格内容对齐方式 tr：当前行内容的对齐方式 table：表示单元格的对齐方式
    bgcolor背景颜色，table，tr，td
    background背景，table，tr，td，建议在CSS中设置。background-size: 100%;
    colspan合并列
    rowspan合并行
    text-align设置文本的水平对齐。可选值：top、baseline、middle、bottom。
    vertical-align设置文本的垂直对齐。
    border-spacing边框间距
    border-collapsecollapse：合并边框，separate：不合并边框，在CSS中设置。

    # form
    属性属性值说明
    actionURL表单数据的提交地址
    method规定如何发送表单数据
    target_blank _self framename规定在何处打开 action URL
    enctype文件上传之前进行编码
    name用于指定表单的名称，以区分同一个页面中的多个表单。
    method的说明：
    get：声明本次请求的目的是从服务器获取数据。
    post：声明本次请求的目的是向服务器推送数据。
    两者区别：
    1. 安全性：get请求在地址栏显示信息，不安全。
    2. 提交数据的长度不同：get不超过255个字符，post不受限制。
    3. 缓存机制：get的请求地址会保存在浏览器的访问历史记录中。
    enctype的说明：
    值描述
    application/x-www-form-urlencoded在发送前编码所有字符（默认），使用URL编码方式进行编码，服务器通过URL方式解码即可，表单中不需要传递文件时使用
    multipart/form-data不对字符编码。因此在服务器端需要进行特殊的解码。 在使用包含文件上传控件的表单时，必须使用该值。
    text/plain空格转换为 "+" 加号，但不对特殊字符编码。

    # 三种引入方式
    内嵌式：
    <!--内嵌样式表-->
<div id="app" style="border: 1px dashed;">
	Hello World!
</div>

    内联式：
    <style>
	/*内联样式表*/
	#app {
		width: 100px;
		height: 100px;
		background-color: greenyellow;
	}
</style>

    外联式：
    <!--外部样式表-->
<link rel="stylesheet" type="text/css" href="outer.css" />

    属性作用
    rel定义当前文档与被链接文档之间的关系，在这里需要指定为“stylesheet”，表示被链接的文档是一个样式表文件。
    type定义所链接文档的类型，在这里需要指定为“text/css”，表示链接的外部文件为CSS样式表。都可以省略
    href定义所链接外部样式表文件的URL，可以是相对路径，也可以是绝对路径。

    # 属性选择器
    <style>
   /*
        * 为所有具有title属性的p元素，设置一个背景颜色为黄色
        * title属性,鼠标移入显示内容
        * 属性选择器
        *    作用：可以根据元素中的属性或属性值来选取指定元素
        *    语法：
        *      [属性名] 选取含有指定属性的元素
        *      [属性名="属性值"] 选取含有指定属性值的元素
        *      [属性名^="属性值"] 选取属性值以指定内容开头的元素
        *      [属性名$="属性值"] 选取属性值以指定内容结尾的元素
        *      [属性名*="属性值"] 选取属性值以包含指定内容的元素
   */
   p[title] {
       background-color: yellowgreen;
   }
   p[title="hello"] {
       background-color: yellow;
   }
   p[title^="ab"] {
       background-color: yellow;
   }
   p[title$="c"] {
       background-color: turquoise;
   }
   p[title*="c"] {
       color: red;
   }
</style>

    # 选择器的优先级
    <!DOCTYPE html>
<html>

	<head>
		<meta charset="UTF-8">
		<title></title>
	</head>
	<style>
		/*
		 * 当使用不同的选择器，选中同一个元素时并且设置相同的样式时，
		 *  这时样式之间产生了冲突，最终到底采用哪个选择器定义的样式，由选择器的优先级（权重）决定
		 *  优先级高的优先显示。
		 * 
		 * 优先级的规则
		 *      内联样式 ， 优先级  1000
		 *      id选择器，优先级   100
		 *      类和伪类， 优先级   10
		 *      元素选择器，优先级 1 
		 *      通配* ，    优先级 0
		 *      继承的样式，没有优先级
		 * 
		 * 当选择器中包含多种选择器时，需要将多种选择器的优先级相加然后在比较，
		 *  但是注意，选择器优先级计算不会超过他的最大的数量级，如果选择器的优先级一样，则使用靠后的样式。对于全是类选择器，元素的路径要写全。
		 *  并集选择器的优先级是单独计算
		 *  div , p , #p1 , .hello{}    
		 * 
		 *  可以在样式的最后，添加一个!important，则此时该样式将会获得一个最高的优先级，
		 *  将会优先于所有的样式显示甚至超过内联样式，但是在开发中尽量避免使用!important
		 * 
		 */
		* {
			font-size: 30px;
		}
		
		p {
			font-size: 20px;
		}
		
		#p2 {
			background-color: yellowgreen;
		}
		/* p和#p2之间没有空格：交集选择器 */
		
		p#p2 {
			background-color: red;
		}
		
		.p3 {
			color: green;
		}
		
		.p1 {
			color: green;
			background-color: greenyellow !important;
		}
	</style>
	</head>

	<body>
		<p class="p1 p3" id="p2" style="background-color: orange;">我是一个段落
			<span>我是p标签中的span</span>
		</p>
	</body>
</html>
